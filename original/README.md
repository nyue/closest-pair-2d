# ClosestPair2D
This algorithm will find the closest pair of points in a list of 2 dimensional points.

## Requirements

  - cmake
  - gnuplot

## Installation

    git clone git@github.com:ThangMinhCao/ClosestPair2D.git
    cd ClosestPair2D
    cmake .
    make

  Then, to run the program:

    ./2d_closest_pair
